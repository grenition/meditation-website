Список описаний

HTML
Язык гипертекстовой разметки, используемый для создания веб-страниц.
CSS
Язык стилей, используемый для оформления и визуального представления веб-страниц.
JavaScript
Язык программирования, используемый для создания интерактивных элементов на веб-страницах.
Frontend
Часть веб-разработки, связанная с созданием пользовательского интерфейса и взаимодействием с пользователем.
Обычно включает работу с HTML, CSS и JavaScript.
Backend
Часть веб-разработки, связанная с серверной логикой, базами данных и обработкой данных.
Отвечает за взаимодействие сервера с клиентом и хранение данных.
